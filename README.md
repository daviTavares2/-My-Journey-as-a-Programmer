# 📖 Minha Jornada como Programador

Este repositório é o registro da minha evolução como desenvolvedor.
Aqui estão reunidos todos os exercícios, projetos e anotações
que produzi ao longo dos meus estudos — do zero ao desenvolvimento web completo.

---

## 📁 Estrutura do Repositório

### 🎨 HTML e CSS
Exercícios práticos feitos por conta própria para consolidar
os fundamentos de marcação e estilização web.
- Estrutura semântica com HTML5
- Estilização e layouts com CSS3
- Prática de seletores, flexbox, responsividade e muito mais

### ⚡ JavaScript
Arquivos e exercícios focados na lógica de programação
e manipulação do comportamento das páginas.
- Fundamentos da linguagem
- Manipulação do DOM
- Exercícios progressivos de lógica

### 🌐 Desenvolvimento Web
Todo o conteúdo do curso completo de Desenvolvimento Web
realizado na **Udemy** — projetos guiados, desafios e anotações.
- Projetos práticos do curso
- Integração de HTML, CSS e JavaScript
- Boas práticas de desenvolvimento

---

## 🎯 Objetivo

Este repositório não é apenas um portfólio —
é um **diário de aprendizado**. Cada pasta, cada arquivo,
representa um passo na minha trajetória como programador.
